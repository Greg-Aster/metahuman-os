
Command	Description
pnpm emu	Start emulator, install APK, launch app
pnpm emu:start	Same as above (explicit)
pnpm emu:install	Just update APK on running emulator
pnpm emu:logs	Watch app logs (logcat)
pnpm emu:stop	Stop the emulator
pnpm emu:status	Check status (shown above)
Quick workflow:
cd apps/mobile
pnpm emu          # Start everything
pnpm emu:logs     # Watch for errors
Then open chrome://inspect in Chrome to debug the WebView.
